Questboard

This was my project for the same Software Development class as above. It is designed to be a task distributor used by teachers in order to disseminate tasks to their students. A teacher can create a Questboard for a specific topic in the course, and assign the number of stars needed to get full marks on any particular quest. The teacher can then post quests in the Questboard where up to three students can call dibs and earn stars for the questboard.
